# Gym-management-system

## 📌 Project Description
A web-based Gym Management System developed using PHP and MySQL to manage gym members, trainers, fees, and workout plans efficiently.

---

## 🚀 Features
- 👤 Member Registration & Management  
- 🧑‍🏫 Trainer Management  
- 💳 Fee / Payment Tracking  
- 📅 Workout Schedule Management  
- 🔐 Admin Login System  
- 📊 Dashboard for overview  
- ✏️ CRUD Operations (Create, Read, Update, Delete)

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: XAMPP  

---

## 📂 Project Structure

/gym-management
│── /admin
│── /user
│── /database
│── index.php
│── config.php


---

## ⚙️ Installation Steps

1. Install XAMPP  
2. Move project folder to:

C:/xampp/htdocs/

3. Start Apache & MySQL  
4. Open phpMyAdmin and import database  
5. Run project in browser:

http://localhost/gym-management
